Setting up Git
==============

In the book, I showed you can use `git <http://git-scm.com/>`_ to
start managing your projects. Installing it on Fedora Linux is as easy
as executing a single command, ``yum -y install git``. On Windows,
using `GitHub for Windows <http://windows.github.com/>`_ is an easy way to start using ``git`` and
also hosting your projects on `GitHub <http://github.com>`_. The following two screencasts
demonstrate how you can setup GitHub for Windows.
